Cliente

(Índice)

São os usuários comuns do software ou site, você os seres humanos , a parte que acessamos quando entramos nesses sites, quando acessamos o Google, Facebook ou até mesmo o Github ou a pagina da Dio. Essa estrutura foi desenvolvida para o usuário final que está rodando em um servidor.

Servidor

(Índice)

São os computadores que processão os dados e as informações programados pelos programadores e disponibilizados em uma rede mundial.

Tipos de servidor

(Índice)

Há diversos tipos de servidores e são eles:

  1. Arquivos
  2. Segurança
  3. Streaming
  4. E-mail
  5. Web

Hospedagem em sites

(Índice)

Todo site ele é hospedado em algum lugar, seja em uma rede interna que você tenha montato dentro da sua casa ou então a rede mundial dos computadores. A hospedagem é em "computador" que está ligado na rede mundial e possui um custo mensal ou anual de acordo com o tipo de servidor que ele é e a necessidade do site/software.

Linguagem de programação

(Índice)

Todo o programa que roda no seu computador, telefone ou qualque outro equipamento eletronico foi desenvolvido com uma linguagem para desempenhar a função que ele exerce, como um manual de instrução escrito por nós humanos para que o computador te entenda e execute ou demonstre o que você está pedindo

HTML

(Índice)

Podemos chama-lo de marcador de grandes textos, não é considerado bem uma linguagem de programação mas sim uma escrita que pode estar ligada a uma linguagem para ser exibida em um navegador e precisa ser categorizada e organizada para ter um bom desenvolvimento.

Estrutura básica

(Índice)

O HTML possui uma estutura padrão que é desenvolvida para o entendimento de como sera demonstrado no texto. Essa estrutura utilizão de artificios,tags, que são os nomes dados para cada função e modo de exibição.

Tags

(Índice)

São usadas para conversar com o navegador como será a estrutura do seu texto e possui 2 tipos, as que precisam de fechamento ou não.

Atributos

(Índice)
Os atributos são usados para personalizar as tags, modificando sua estrutura ou funcionalidade. Igualmente, os atributos são utilizados para atribuir uma classe ou id a um elemento.
